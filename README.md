## `j2_library` 라이브러리

<br />

---

### 0. `j2_library` 소개

- `j2_library`는 `C++` 표준을 기반으로 작성된 크로스 플랫폼 라이브러리입니다.
	- 현재 `C++17` 표준 사용
	- `Windows`, `Linux` 지원 (`MacOS` 지원 예정)
	- `cmake`, `ninja` 빌드 시스템 사용

- 홈페이지
	- https://github.com/JayTwoLab/jaytwo-library

<br />
